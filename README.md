## Parrot

Turn Mycroft into a echoing parrot!

## Description

    "Hey Mycroft, start parrot"
    "hello"
    hello
    "what"
    what
    "who are you"
    who are you
    "Stop parrot"

NOTE: This illustrates the converse() method

## Examples
* "start parrot"
* "stop parrot"
